# 🚀 НАЧНИТЕ ЗДЕСЬ

## ⚠️ Ошибка "Не удалось загрузить JavaScript"

Эта ошибка означает, что проект еще не собран. Следуйте инструкциям ниже.

## ✅ Автоматическая установка (РЕКОМЕНДУЕТСЯ)

**Самый простой способ** - запустите PowerShell скрипт:

```powershell
# Разрешите выполнение скриптов (один раз, если нужно)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# Запустите скрипт
.\setup-and-build.ps1
```

Или с обходом политики выполнения:

```powershell
powershell -ExecutionPolicy Bypass -File .\setup-and-build.ps1
```

Скрипт автоматически:
1. ✅ Установит Node.js (если нужно)
2. ✅ Установит зависимости проекта
3. ✅ Соберет проект
4. ✅ Проверит готовность

## ✅ Ручная установка (если автоматическая не работает)

### 1. Установите Node.js

- Скачайте: https://nodejs.org/ (выберите LTS версию)
- Установите с настройками по умолчанию
- Перезапустите терминал/PowerShell

Подробнее: [INSTALL_NODEJS.md](./INSTALL_NODEJS.md)

### 2. Установите зависимости

Откройте терминал в папке проекта и выполните:

```bash
npm install
```

### 3. Соберите проект

```bash
npm run build
```

После этого появится папка `dist/` с готовыми файлами.

### 4. Загрузите расширение в Chrome

1. Откройте `chrome://extensions/`
2. Включите **"Режим разработчика"** (справа вверху)
3. Нажмите **"Загрузить распакованное расширение"**
4. **ВАЖНО:** Выберите папку **`dist/`** (не корневую папку проекта!)

## 📋 Проверка готовности

После сборки выполните:

```bash
npm run check
```

Это проверит, что все файлы на месте.
