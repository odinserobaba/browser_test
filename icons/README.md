# Icons

Поместите сюда иконки расширения:

- `icon16.png` - 16x16 пикселей
- `icon48.png` - 48x48 пикселей  
- `icon128.png` - 128x128 пикселей

Вы можете создать иконки используя любой графический редактор или онлайн генератор иконок.

Временное решение: создайте простые PNG файлы с цветным фоном и текстом "APR" (AI Playwright Recorder).
