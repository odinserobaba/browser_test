# 🚀 Быстрый старт

## Установка и настройка за 5 минут

### Шаг 1: Установка зависимостей

```bash
npm install
```

### Шаг 2: Сборка расширения

```bash
npm run build
```

### Шаг 3: Загрузка в Chrome

1. Откройте `chrome://extensions/`
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку проекта

### Шаг 4: Создание иконок (опционально)

Создайте три PNG файла в папке `icons/`:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

Или используйте временные иконки с любого онлайн генератора.

### Шаг 5: Настройка API ключа

1. Откройте расширение (клик на иконку)
2. Введите OpenAI API ключ
3. Нажмите "Сохранить настройки"

### Шаг 6: Готово!

Теперь вы можете:
- Начать запись действий на любой странице
- Генерировать Playwright тесты с AI-локаторами
- Сохранять тесты в выбранную папку

## Первый тест

1. Откройте любой сайт (например, `https://example.com`)
2. Нажмите "Начать запись" в popup
3. Выполните несколько действий (клики, ввод текста)
4. Нажмите "Остановить"
5. Нажмите "Сгенерировать тест"
6. Запустите сгенерированный файл: `python test_generated.py`

## Troubleshooting

**Ошибка компиляции TypeScript:**
```bash
npx tsc --noEmit
```

**Расширение не загружается:**
- Проверьте, что папка `dist/` существует
- Убедитесь, что `manifest.json` в корне проекта

**API ключ не работает:**
- Проверьте баланс на OpenAI аккаунте
- Убедитесь, что ключ начинается с `sk-`
