# Contributing to AI Playwright Recorder

Спасибо за интерес к проекту! Мы рады вашему вкладу.

## Как внести вклад

### Сообщение об ошибке

Если вы нашли баг:
1. Проверьте, не была ли ошибка уже зарегистрирована в Issues
2. Создайте новый Issue с описанием:
   - Шаги для воспроизведения
   - Ожидаемое поведение
   - Фактическое поведение
   - Версия браузера и расширения
   - Скриншоты (если применимо)

### Предложение новой функции

1. Создайте Issue с меткой `enhancement`
2. Опишите проблему, которую решает функция
3. Предложите реализацию (если есть идеи)

### Pull Request

1. Fork репозиторий
2. Создайте feature branch (`git checkout -b feature/amazing-feature`)
3. Внесите изменения
4. Убедитесь, что код компилируется (`npm run build`)
5. Проверьте типы (`npx tsc --noEmit`)
6. Commit с понятным сообщением
7. Push в ваш fork
8. Откройте Pull Request

### Code Style

- Используйте TypeScript strict mode
- Следуйте ESLint правилам
- Добавляйте JSDoc комментарии к публичным функциям
- Используйте понятные имена переменных

### Тестирование

Перед отправкой PR убедитесь:
- [ ] Код компилируется без ошибок
- [ ] Нет TypeScript ошибок
- [ ] Расширение работает в Chrome
- [ ] README обновлен (если нужно)

Спасибо за ваш вклад! 🎉
